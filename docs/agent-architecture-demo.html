Local Validation Loop

src/services/local-fix-validator.ts — Apply, test, iterate, push.
What it receives from the agent pipeline
The approved changes array from Fix Generation Agent:
changes[0].oldCode → find this in the local file
changes[0].newCode → replace with this

If the test fails, testResult.logs (last 20K chars of stdout/stderr) are fed back to the repair agent as previousAttempt.validationLogs, and the entire 5-agent pipeline runs again. 
1. Find oldCode in local file → replace with newCode
2. Run: npx wdio wdio.conf.ts --spec {spec} -b chrome {url} -t sauce
3. If PASS → git push + create PR
4. If FAIL → reset files, feed logs back, run all 5 agents again
Model: gpt-5.3-codex · API: OpenAI Responses API · Runtime: Node.js 24 · Source: adept-at/adept-triage-agent
